# SuryaHub E-Commerce Revenue Optimization

A Project from Harisenin.com Bootcamp: Data-driven customer segmentation and loyalty program transformation project designed to reverse revenue stagnation and optimize marketing ROI.

## Description
This project analyzes a dataset of 499 customers and Rp2.1B+ in total transactional revenue from June 2025. By integrating SQL data extraction, Python data processing, and interactive Looker Studio dashboards, this analysis uncovers critical user behavior paradoxes and establishes a hyper-targeted customer segmentation model.

## Problem
* **Revenue Stagnation:** SuryaHub experienced flatlining revenue per customer despite positive transaction volume growth.
* **Inefficient Marketing Spend:** Marketing budgets were allocated via generic mass discounts rather than high-conversion channels.
* **Membership Red Flag:** Active loyalty program members generated lower purchase frequencies and lower average spending than non-members.

## Solution
* **Data-Driven Analysis:** Evaluated brand loyalty metrics against discount usage using SQL and Python to pinpoint margin-eroding promotional habits.
* **Goldmine Segment Discovery:** Isolated high-value customer clusters by filtering transactional data across location, gender, income level, and buying intent.
* **Interactive Dashboarding:** Built an omnichannel analytics dashboard in Looker Studio to visualize complex customer journeys across Online, In-Store, and Mixed channels.

## Recommendation
* **Revamp Membership Benefits:** Shift budgets from mass discounts to experiential rewards like automated free shipping and surprise physical merchandise.
* **Pivot Content Strategy:** Transform social media content from hard-selling to product education guides to capture high-value rational buyers.
* **Bridge Retail to Digital:** Deploy in-store QR code frameworks at physical cash registers to convert habitual retail shoppers into tracked digital members.

## Impact
Formulated a structured, 4-week personalized marketing roadmap for July 2025 that transitions SuryaHub from a discount-heavy framework to a value-first retention model, directly targeting an increase in average revenue per customer.
